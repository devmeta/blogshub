Getting started

1. Cloning caldero

Git clone

git clone git@github.com:devmeta/caldero.git

2. Dump SQL schema

Create a database named caldero and run ... 

SET NAMES utf8;
SET time_zone = '+00:00';
SET foreign_key_checks = 0;
SET sql_mode = 'NO_AUTO_VALUE_ON_ZERO';

DROP TABLE IF EXISTS `files`;
CREATE TABLE `files` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `name` varchar(255) NOT NULL DEFAULT '',
  `title` varchar(255) NOT NULL DEFAULT '',
  `content` text NOT NULL,
  `tag` varchar(20) NOT NULL DEFAULT 'post',
  `position` varchar(20) NOT NULL DEFAULT '1',
  `parent_id` int(11) NOT NULL DEFAULT '0',
  `file_size` int(20) NOT NULL DEFAULT '0',
  PRIMARY KEY (`id`)
) ENGINE=InnoDB DEFAULT CHARSET=latin1;

3. Check your DB access config

Open your config file


$ nano /var/www/caldero/vendor/caldero/includes/config.php

Edit with your database credentials


define('DBHOST','localhost');
define('DBUSER','root');
define('DBPASS','********');
define('DBNAME','caldero');

4. jQuery

Hook caldero jquery route handlers



// Applies just when post tag is displayed

/* route handlers */

$(document).on('post-tag','body',function(){
  $('.slick-std').slick({  
    dots: true,

jQuery onshow handlers

Hook caldero on every render


$(document).on('display','body',function(){
  // Applies on every page displayed
});

Hook caldero when one view is displayed with a handler


$(document).on('home','body',function(){
  // Applies just when home page is displayed
});

5. Nginx settings

Create a new nginx entry like this


server {

        root /var/www/caldero;

        index index.php index.html index.htm;

        server_name caldero;

        location / {
                try_files $uri $uri/ /index.php$is_args$args;
        }

        location ~ \.php$ {
                fastcgi_pass unix:/var/run/php5-fpm.sock;
                fastcgi_index index.php;
                include fastcgi_params;
        }
}
